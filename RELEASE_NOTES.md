# MDive v1.0.1

MDive（墨潜）是一个以工作区文件夹为核心的本地 Markdown 查看与编辑工具，适合配合 VS Code / Claude Code 作为实时预览器，也可以独立完成 Markdown 写作、预览、搜索和导出。

## 版本亮点

- 支持 Markdown / 图片 / PDF 等本地文件内嵌预览
- 支持源码 / 分栏 / 预览三档视图切换
- 支持 Mermaid 图表渲染、文档大纲、同步滚动和全文搜索
- 支持按 A4 分页导出 PDF，并通过 JPEG 压缩控制文件体积
- 新建未命名 Markdown 文件关闭时会触发保存确认，避免内容丢失
- 标签栏在文件过多时自动压缩，鼠标悬停显示完整文件名
